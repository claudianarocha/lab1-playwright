# 🎭 Laboratório de Automação de Testes - Login com Playwright

Este repositório contém um ambiente de login desenvolvido para prática de automação de testes end-to-end (E2E) utilizando a ferramenta Playwright. <br>
<br>

## 🌐 Demonstração Online

## 🧪 Sobre o Projeto

Este projeto foi criado com o objetivo de simular cenários reais de testes automatizados em uma aplicação web, com foco em:

* Validação de autenticação (login)
* Testes positivos e negativos
* Verificação de mensagens de erro
* Simulação de fluxo do usuário <br>
<br>

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Playwright
* JavaScript
* Vercel (Deploy) <br>
<br>

## 🧠 Habilidades Demonstradas

* Automação de testes E2E
* Escrita de cenários de teste
* Validação de fluxo de login
* Organização de projeto para testes automatizados <br>
